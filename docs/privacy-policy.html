Privacy Policy for SubTerrania

Last updated: May 13, 2026

Overview

SubTerrania is a single-player, offline turn-based strategy game developed by Atlyn (Garrett Hamers). The game itself works fully offline. Starting in version 1.0.6, the app sends a small amount of anonymous diagnostic and gameplay-outcome telemetry to Azure Application Insights when an internet connection is available, so we can detect crashes and understand which difficulties / characters / maps are most popular. We never collect personal information.

What We Collect

Diagnostic telemetry (Azure Application Insights)

When your device has an internet connection, SubTerrania sends the following non-personal events to Azure Monitor (operated by Microsoft):

  • app_start — every time the app launches
  • game_started — with the chosen difficulty, character, and map
  • game_won / game_lost — with difficulty, character, map, turn number, and victory points earned
  • feedback_sent — when you tap "Send feedback" in the in-app settings
  • Uncaught exceptions (crashes) including the stack trace, exception type, and message — but no game contents

Each event also includes the app version (e.g. 1.0.6), Android API level (e.g. 34), and device model name (e.g. "Pixel 6"). A randomly-generated session-instance ID is included with each batch but is not persisted across launches and cannot be tied to any account or device fingerprint.

What we do NOT collect

  • Your name, email, account identifier, or any contact information
  • Your IP address (Azure ingestion logs the connection IP for abuse prevention but we never query or analyse it)
  • Your game-save contents, board layouts, or any free-form text you might type
  • Any persistent device or installation identifier (no advertising ID, no Android ID, no install UUID)
  • Your contacts, location, camera, microphone, files, or browsing history

Permissions

The app requests only two permissions:

  • VIBRATE — for haptic feedback during gameplay. No data collection.
  • INTERNET — used solely to send the telemetry described above to Azure Application Insights. The app never accepts inbound connections, and gameplay continues to work fully if the network is unavailable (telemetry is silently dropped).

Third-Party Services

SubTerrania uses one third-party service:

  • Microsoft Azure Application Insights — receives the diagnostic telemetry described above. See Microsoft's privacy statement for how Microsoft handles data sent to its services.

The app does not integrate with any analytics SDK, advertising network, social-media SDK, or third-party login provider.

Data Retention

Telemetry is retained for 90 days in our Azure workspace and then automatically deleted. Because we don't collect any identifier that ties events back to you, there is no way for us to delete a specific user's data on request — but there is no personal data to delete.

Opting Out

To stop the app from sending any telemetry, simply put your device in airplane mode or revoke the app's network access in Android settings. The game itself will continue to work normally; only the diagnostic events will stop. (A future release may add an in-app opt-out toggle.)

Children's Privacy

SubTerrania is rated for all ages and does not target children under 13. We do not knowingly collect personal information from anyone, including children. The diagnostic telemetry described above does not include any data that could identify a child or any specific user.
